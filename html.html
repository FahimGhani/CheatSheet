HTML Notes

Tag Name Description
< DOCTYPE html > DOCTYPE The Tag tels the browser what type of information to expect.
< html > ... < html /> HTML Contains all HTML elements except for < DOCTYPE html >.
< head > ... < head /> Head It is used to define the head portion of the document which contains information related to the document.
< body > ... < body /> Body It used to define the main content present inside an HTML page.
< header > ... < header /> Header It is used to separate headings and subheadings inside the Body Tag.
< main > ... < main /> Main It specifies the main content of a document.
< footer > ... < footer /> Footer It used to define a footer of HTML document.
< h1 > ... < h1 /> Heading It indicates a title on a website.
< p > ... < p /> Paragraph It is used to defines a paragraph/phrases.
< div > ... < div /> Division It is used in HTML to make divisions of content in the web page.
< em > ... < em /> Italic Emphasis Turns The wrapped text into italic.
< strong > ... < strong /> Bold Emplasis It is used to define text with strong importance. The content inside is typically displayed in bold.
< nav > ... < nav /> Navigation It is used for declaring the navigational section in HTML documents.
< article > ... < article /> Article An article is used to write something that can be only found written between this tag. Ex: Forum Post, Blog Post or News Story.
< section > ... < section /> Section It is defines the section of documents such as chapters, headers, footers or any other sections.
< aside > ... < aside /> Aside It is used to defines some content aside from the content it is placed in.
< figure > ... < figure /> Figure It specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.
< figcaption > ... < figcaption /> Figure Caption. It is a caption or legend describing the rest of the contents or the Figure Tag.
< audio > ... < audio /> Audio It is used to embed sound content in a document.
< video > ... < video /> Video It is used to embed video content in a document.
< embed > ... < embed /> Embed It is used for embedding external applications like gif.
< br /> Break Line It is used to produces a line break in text.
< ul > ... < ul /> Unordered listed It is used to create a list that uses bullet points.
< ol > ... < ol /> Ordered List It is used to create a list that uses Number System.
< img /> Image it is used to instructions on how to display an image
< li > ... < li /> List Item It is used to represent an item in a list
< a > ... < a /> Anchor It is used to create a hyperlink on the webpage.
< !-- ---> Comment Out It is used to insert comments in the source code.
< table > ... < table /> Table It is used to defines the overall table.
< tr > ... < tr /> Table Row It is used to create a table row.
< thead > ... < thead /> Table Head It is used to define the header of the table.
< tbody> ... < tbody /> Table Body It is used to define the body of the table.
< tfoot> ... < tfoot/> Table Footer It is used to define the footer of the table.
< td > ... < td /> Table Data It is used to define the data in the table.
